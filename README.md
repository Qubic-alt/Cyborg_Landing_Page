## ✨ Features
- **Neon UI/UX:** A dark-themed aesthetic with neon cyan and purple accents.
- **Interactive Cursor:** A custom-built reactive cursor that follows user movement.
- **Scanline & Grid Effects:** Modern CSS techniques used to create a "digital screen" feel.
- **Scroll Animations:** Smooth entry animations powered by AOS (Animate On Scroll).
- **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop screens.
- **Interactive Cards:** Hover-aware cards with glassmorphism effects.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom animations, Flexbox/Grid, and Neon effects.
- **JavaScript:** Custom cursor logic and interactive status bar.
- **Lucide Icons:** High-quality tech-focused iconography.
- **AOS Library:** For scroll-triggered animations.
